## Installation

1. Go to the [Release 1.0 page](https://github.com/ChinthanaBhat/Flappy_Bird/releases/tag/1.0).
2. Download the `.exe` installer file provided in the release assets.
3. Double-click the downloaded file and follow the on-screen setup instructions to install the game on your system.
4. Launch the game from your Start Menu or Desktop shortcut!

## How to Play

1. Start the game.
2. Press the **Spacebar** to make the bird flap and fly upwards.
3. Gravity will constantly pull the bird down, so keep flapping to stay in the air.
4. Carefully navigate through the gaps in the green pipes.
5. You earn one point for every set of pipes you successfully pass.
6. If you touch a pipe or the ground, it's game over!
7. Try to beat your high score!

Enjoy the game!
